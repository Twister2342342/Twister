Гайды/инструкция — Nigthix + моды

Как использовать

1. Откройте `index.html` в браузере.
2. Кнопка "Скачать Nigthix + моды" ведёт на внешний файл: https://workupload.com/file/4HYL6tAgE7M
3. Следуйте инструкции на странице в разделе "Инструкция по установке".

Файлы

- index.html — основной лендинг
- styles.css — стили с плавными градиентами и анимациями
- script.js — логика модального окна и формы

Примечание

Форма на странице демонстрационная и не отправляет данные на сервер. Связь для помощи: https://t.me/+AzkhvHOigmUwZWMy
