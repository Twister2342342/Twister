Aurora — современный лендинг на тему "Майнкрафт чит"

Как использовать

1. Откройте `index.html` в браузере (двойной клик или `File → Open`).
2. Форма запроса доступа — демонстрационная и не отправляет данные на сервер.

Файлы

- index.html — основной лендинг
- styles.css — стили
- script.js — простая логика модального окна и формы

Примечание

Этот репозиторий содержит только фронтенд-лендинг. Никаких инструкций по созданию или распространению читов не включено.
